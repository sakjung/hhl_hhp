# Hiphople & Hiphopplaya

## Introduction

[`Hiphople`](http://hiphople.com/)와 [`Hiphopplaya`](https://hiphopplaya.com/) 는 대한민국의 힙합 커뮤니티 사이트 이다. 본 프로젝트에서는 Rselenium과 Selenium을 활용하여 다이나믹하게 변화하는 두 웹사이트에서 게시물과 댓글들을 크롤링하는 법에 대해서 시도하였다.
